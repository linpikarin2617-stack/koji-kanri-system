# 工事管理システム

ブラウザで動作するシンプルな工事管理Webアプリです。インストール不要で、`index.html` をダブルクリックするだけで使えます。

## 主な機能

- **ダッシュボード** — 年度別の工事一覧と進捗確認
- **工事台帳** — 工事の登録・編集・ステータス管理
- **実行予算書** — バージョン管理付きの予算作成
- **見積書 / 外注依頼書 / 注文書** — 各種書類の作成と承認フロー
- **原価管理** — 予算と実績のコスト比較
- **精算書** — 工事完了後の最終精算
- **設定** — 社員マスタ・承認ワークフローの設定

## 使い方

1. `index.html` をブラウザで開く
2. 上部タブでメニューを切り替える
3. データはブラウザの `localStorage` に自動保存される

## データのバックアップと復元

- **保存**: 「データ保存」ボタンでJSONファイルをエクスポート
- **復元**: 「データ読込」ボタンでJSONファイルをインポート

## 動作環境

`localStorage` と `FileReader API` をサポートするモダンブラウザ（Chrome / Edge / Firefox 最新版）
